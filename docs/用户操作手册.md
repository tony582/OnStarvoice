# StarVoice 星语 · 用户操作手册(运营版)

面向品牌方运营人员。讲**怎么用**——采集、看分诊、跟客资、出报告;不涉及代码。

---

## 一、平台能帮你做什么

用 Chrome 扩展采集 **小红书 / 抖音 / 微博** 的作品和评论 → 后台自动用 AI 判断**相关性、情感、负面、购买意向** → 帮你:
- **内容分诊**:把涉及品牌的内容按风险排好,逐条处置;
- **销售客资**:从评论里挑出有购买/咨询意向的人;
- **预警 / 工单 / 事件**:负面及时报、转处理、聚类追踪;
- **分析报告**:趋势、情感结构、日/周/月报。

---

## 二、开始前准备(一次性)

1. **装扩展**:解压收到的扩展包 → Chrome 打开 `chrome://extensions` → 右上角开「**开发者模式**」→「**加载已解压的扩展程序**」→ 选解压出来的文件夹 → 建议固定到工具栏。
2. **登录后台**:浏览器打开 `https://voice.minilife.online/admin`,用分配的账号或激活码登录。
3. **（管理员)配置好基础项**(后台右上角 → 系统设置 / 管理):
   - **AI 模型**:填提供商 + 模型 + API Key。国内请用 **DeepSeek** 或 **通义千问**(`qianwen` + `qwen3.x-flash`);⚠️ 别用 Gemini(国内连不上)。
   - **品牌监控范围**:决定 AI 把哪些内容判为"相关"。安吉星默认是**整个上汽通用**(别克/凯迪拉克/雪佛兰 + 车机壁纸/周边)。
   - **官方账号**:登记你们的官方号,用于识别"官方已回复"。
   - **监控订阅**(关注博主):配关键词/博主,系统定时提醒。

## 三、怎么采集数据(用扩展)

在小红书/抖音/微博页面点扩展图标,侧栏顶部有几个入口:

| 入口 | 用途 | 要评论就这样 |
|---|---|---|
| **作品页** | 打开一篇作品详情页 → 「采集作品数据」 | 打开「**附加采集评论**」+ 填「**评论加载上限**」(如 1000),评论一起采**最完整** |
| **账号页** | 打开博主主页 → 采该博主的作品 | 同上 |
| **搜索页** | 搜关键词 → 批量采搜索结果 | 开「**采集增强**」会逐条点开详情补采正文/互动/评论/客资 |
| **对标监控** | 对标账号库管理 | — |

采完点「**同步后台**」即可。

**小贴士**
- 要**完整评论**:优先用「作品页」打开作品采,或搜索页开「采集增强 + 附加评论」。
- 评论同步后会**先按规则显示出来**,后台 AI 再**逐条精炼**(情感/负面/购买意向);分诊页顶部「**评论 AI 分析中**」进度条跑完就全好了——**不用一直等,评论不会丢**。

## 四、后台怎么用

顶部切换两个工作区:**舆情风控**(监测处置)/ **内容创意**(选题爆款)。

### 舆情风控

- **指挥中心**:全局总览——情感结构、分平台风险、待处理、最新命中。每天先看这里。
- **舆情工作台**(核心):
  - **内容分诊**:所有相关作品的处置台。默认**最新发布在前**;点「发布时间 / 互动」列头可**排序**;可按**平台 / 情感 / 风险信号 / 处置状态**筛选;每条可「**转工单**」、归档、标误报。看板视图可拖卡改状态。
  - **评论分诊 / 销售客资**:AI 从评论里挑出的购买/咨询意向线索。
  - **已转工单**:转出去的工单跟踪反馈。
- **舆情处理**:工单与事件的集中处置。
- **关注博主**:监控订阅(定时采某关键词/博主的新内容)。
- **销售客资**:销售线索汇总跟进。
- **分析与报告**:趋势、情感、日/周/月报,可导出。
- **数据底座**:全部原始记录,可检索回溯。

### 内容创意
内容总览 · 赛道大盘 · 爆款拆解 · 对标账号库 · 选题与扩词 · 内容复盘。

---

## 五、典型的一天

1. **采集**:扩展里搜关键词批量采 + 打开几条重点作品采评论 → 同步。
2. **分诊**:进「内容分诊」,按风险/负面筛,逐条处置(转工单/归档/标误报)。
3. **客资**:进「销售客资」跟进购买意向线索。
4. **复盘**:进「分析与报告」看趋势、导报告。

## 六、常见问题

| 问题 | 说明 |
|---|---|
| 评论显示 0 / 没出来 | 后台 AI 还在精炼(看顶部进度条),稍等即可;若这条是搜索页采的且没开「附加评论」,用「作品页」打开重采。 |
| 评论采不全 | 抖音的「评论数」**含回复(子评论)**;主评论已采全,回复的自动展开还在迭代中。 |
| 发布时间不对 | 已修复;请用**最新扩展**重采(旧数据日期不会自动改)。 |
| 别克/凯迪拉克的内容算不算 | 算。监控范围是**整个上汽通用**,在系统设置的"品牌范围"里配。 |
| 用哪个 AI 模型 | 国内用 **DeepSeek** 或 **通义千问**;**Gemini 在国内连不上**。 |
| 换了扩展没生效 | 改/换扩展后要在 `chrome://extensions` 点 **Reload**,再重新采集。 |
